Privacy Policy / プライバシーポリシー

Last Updated / 最終更新日: 2026-03-30

はじめに / Introduction

本プライバシーポリシーは、Chrome拡張機能「Salesforce ReportTools」（以下「本拡張機能」）におけるユーザーデータの収集、使用、保存、共有について説明します。

This Privacy Policy describes how the Chrome extension "Salesforce ReportTools" (the "Extension") collects, uses, stores, and shares user data.

収集する情報 / Information We Collect

Salesforceセッション情報 / Salesforce Session Information

本拡張機能は、Salesforce APIとの通信を行うために、ブラウザのCookieからSalesforceのセッションID（sid）を読み取ります。このセッション情報は本拡張機能内でのみ使用され、外部サーバーに送信・保存されることはありません。

The Extension reads Salesforce session IDs (sid) from browser cookies to communicate with Salesforce APIs. This session information is used only within the Extension and is never transmitted to or stored on external servers.

Salesforceレポートメタデータ / Salesforce Report Metadata

本拡張機能は、SalesforceのAPI（Analytics API、SOAP Metadata API、Tooling API、sObject Describe API）を使用してレポートメタデータを取得・更新します。これらのデータはすべてユーザーのSalesforce組織とブラウザ間で直接やり取りされ、第三者のサーバーを経由しません。

The Extension uses Salesforce APIs (Analytics API, SOAP Metadata API, Tooling API, sObject Describe API) to retrieve and update report metadata. All data is exchanged directly between the user's Salesforce org and the browser, without passing through any third-party servers.

利用統計情報 / Usage Statistics

本拡張機能は、将来的にGoogle Analyticsを使用して匿名の利用統計情報（利用頻度、機能別使用回数、セッション時間など）を収集する場合があります。これらの情報は個人を特定するものではなく、拡張機能の改善のためにのみ使用されます。

The Extension may collect anonymous usage statistics (usage frequency, feature usage counts, session duration, etc.) via Google Analytics in the future. This information does not identify individuals and is used solely to improve the Extension.

情報の使用目的 / How We Use Information

収集した情報は、以下の目的でのみ使用されます。

Collected information is used only for the following purposes:

  • Salesforceレポートメタデータの表示、編集、エクスポート機能の提供
    Providing report metadata viewing, editing, and export functionality
  • 拡張機能の改善および利用状況の分析
    Improving the Extension and analyzing usage

情報の保存 / Data Storage

  • 拡張機能の設定はブラウザのローカルストレージ（chrome.storage）に保存されます。
    Extension settings are stored in the browser's local storage (chrome.storage).
  • Salesforceから取得したメタデータはブラウザのメモリ上で一時的に処理され、永続的に保存されることはありません。
    Metadata retrieved from Salesforce is temporarily processed in browser memory and is not persistently stored.
  • 本拡張機能は独自のサーバーを持たず、ユーザーデータを外部サーバーに保存しません。
    The Extension does not operate its own servers and does not store user data on external servers.

情報の共有 / Information Sharing

本拡張機能は、以下の場合を除き、ユーザーの情報を第三者と共有しません。

The Extension does not share user information with third parties, except in the following cases:

  • Salesforce API: レポートメタデータの取得・更新のため、ユーザーのSalesforce組織に直接通信します。
    Communicates directly with the user's Salesforce org for retrieving and updating report metadata.
  • 法律に基づく要求がある場合。
    When required by law.

Cookieの使用 / Use of Cookies

本拡張機能は、SalesforceドメインのCookieからセッションIDを読み取ります。これはSalesforce APIへの認証に必要であり、それ以外の目的でCookieを使用することはありません。

The Extension reads session IDs from Salesforce domain cookies. This is required for Salesforce API authentication and cookies are not used for any other purpose.

ユーザーの権利 / User Rights

  • ユーザーはいつでも本拡張機能をアンインストールし、ローカルストレージに保存されたすべてのデータを削除できます。
    Users can uninstall the Extension at any time, which deletes all data stored in local storage.
  • Cookieの管理はブラウザの設定から行えます。
    Cookie management can be done through browser settings.

セキュリティ / Security

本拡張機能はユーザーデータの保護に努めています。すべてのAPI通信はHTTPSを使用して暗号化されます。ただし、インターネット上のデータ転送は100%安全であることを保証するものではありません。

The Extension strives to protect user data. All API communications are encrypted using HTTPS. However, no method of data transmission over the Internet is guaranteed to be 100% secure.

プライバシーポリシーの変更 / Changes to This Privacy Policy

本プライバシーポリシーは予告なく変更される場合があります。変更があった場合は、このページを更新し、最終更新日を変更します。

This Privacy Policy may be changed without notice. If changes are made, this page will be updated and the last updated date will be revised.
